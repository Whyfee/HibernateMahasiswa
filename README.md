# Aplikasi CRUD Mahasiswa
Aplikasi pengelolaan data mahasiswa menggunakan Java, Hibernate, dan JasperReports.  
Aplikasi ini memungkinkan pengguna untuk mengelola data mahasiswa secara efisien dan mencetak laporan ke dalam format PDF.

## Fitur
- Create, Read, Update, Delete (CRUD) data mahasiswa
- Koneksi database menggunakan Hibernate ORM
- Validasi data mahasiswa
- Cetak laporan dengan JasperReports
- Export laporan ke PDF
- Antarmuka GUI menggunakan Java Swing 

## Teknologi
- Java SE
- Hibernate ORM
- MySQL 
- JasperReports
- iReport
- NetBeans 

## Database
**Nama Database:** `db_mahasiswa`

**Tabel:** `mahasiswa`

## Setup
1. Instal JDK dan IDE (NetBeans/IntelliJ)
2. Buat database `db_mahasiswa`
3. Buat tabel `mahasiswa` sesuai struktur
4. Konfigurasi file `hibernate.cfg.xml`
5. Import file laporan `.jrxml` ke JasperReports
6. Jalankan aplikasi dari IDE
